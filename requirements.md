# Project Goal: Elite Gaming News Website

## Core Philosophy
- Startup product feel
- Smooth interactions
- Professional UI polish
- Production-style architecture

## Tech Stack
- React, Tailwind CSS, React Router, Axios, Framer Motion, React Toastify

## Requirements Checklist

### 1. Visual Style
- [ ] Dark gaming theme
- [ ] Gradient accent colors
- [ ] Modern card-based layout
- [ ] Smooth hover effects
- [ ] Micro-animations

### 2. Homepage Features
- [ ] Trending gaming news
- [ ] News cards (Image, Title, Desc (max 120 chars), Date, Read more)
- [ ] Search bar with debounce optimization
- [ ] Category filter system
- [ ] Pagination or infinite scroll

### 3. Detail Page Design
- [ ] Full news article content
- [ ] Large hero image
- [ ] Metadata (author, date)
- [ ] Frontend comment section UI

### 4. Authentication UI (Frontend Only)
- [ ] Professional Login page
- [ ] Professional Register page

### 5. Dark Mode System
- [ ] Custom `useDarkMode` hook
- [ ] Save preference in localStorage
- [ ] Smooth transition animation

### 6. Custom React Hooks
- [ ] `useFetchNews` (fetch data)
- [ ] `usePagination` (pagination logic)
- [ ] `useDebounce` (optimize search)

### 7. Performance Optimization
- [ ] Lazy loading routes (React lazy + Suspense)
- [ ] Lazy load images
- [ ] Skeleton loaders

### 8. State Management
- [ ] Theme management Context
- [ ] Authentication UI state Context

### 9. Responsive Design
- [ ] Mobile-first approach

### 10. Accessibility

### 11. Animations (Framer Motion)
